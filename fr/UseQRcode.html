1-Entrer à la page requise:


Vous pouvez modifier les paramètres à gauche pour générer un code QR personnalisé.

2-Entrez le texte à Générer:

Entrez le contenu à Générer en Code Qr dans la zone de texte suivant.

3-Sélectionner le niveau de correction d'erreur:


Capacité à corriger les erreurs :

Niveau L : environ 7 % de redondance.
Niveau M : environ 15 % de redondance.
Niveau Q : environ 25 % de redondance.
Niveau h : environ 30 % de redondance.

4-Largeur et hauteur:

Entrez Largeur et Hauteur dans le zone de texte ou sélectionner au Barre bleu.

5-Couleur de la ligne et de l'arrière-plan:

Cliquez sur le point noir pour sélectionner le Couleur de la ligne d'un Code QR à Générer ou cliquez sur le point blanc pour sélectionner le Couleur de l'arrière-plan à Générer.

6-Générer le Code QR:

Cliquez sur le bouton "Générer le Code QR" pour générer le Code QR.

7-Enfin, vous pouvez télécharger le Code QR ou imprimer selon votre choix:
